# ChatCore Agent жҢҮеҚ—пјҲдёӯж–ҮзүҲпјү

жң¬ж–Үд»¶жҳҜ `AGENTS.md` зҡ„дёӯж–Үзҝ»иҜ‘зүҲпјҢдё»иҰҒжҸҗдҫӣйЎ№зӣ®з»“жһ„гҖҒд»Јз ҒйЈҺж јеҸҠи®ҫи®ЎжҖқи·Ҝзҡ„жҰӮиҰҒпјҢдҫҝдәҺдёӯж–ҮиҜ»иҖ…дҪҝз”ЁгҖӮ

## еҗҢжӯҘеЈ°жҳҺ
- жң¬ж–ҮжЎЈжҳҜеҜ№ `AGENTS.md`зҡ„з®ҖеҚ•зҝ»иҜ‘пјҢд»Ҙдёӯж–ҮжҸҗдҫӣгҖӮ
- иҜ·еңЁйҳ…иҜ»д»Јз Ғж—¶еҝҪз•ҘжӯӨж–Үд»¶пјҢеҰӮжһңеҜ№жҢҮеҚ—ж–ҮжЎЈжңүдҝ®ж”№пјҢеҝ…йЎ»е…Ҳжӣҙж”№ `AGENTS.md`пјҢз„¶еҗҺе°Ҷдҝ®ж”№еҗҢжӯҘеҲ°жң¬ж–ҮжЎЈгҖӮ

## йЎ№зӣ®жҰӮиҝ°
ChatCore жҳҜдёҖдёӘ .NET еӨҡйЎ№зӣ®и§ЈеҶіж–№жЎҲпјҢжҸҗдҫӣ Twitch е’Ң Bilibili иҒҠеӨ©жңҚеҠЎзҡ„е…ұз”Ёе®ўжҲ·з«Ҝеә“гҖӮдё»иҰҒеҢ…еҗ«пјҡ
- `ChatCore`пјҡеҹәдәҺ .NET Standard 2.0 зҡ„ж ёеҝғеә“пјҢе®һзҺ° Twitch е’Ң Bilibili зӯүжңҚеҠЎпјҢеҢ…еҗ«й…ҚзҪ®дёҺж—Ҙеҝ—еҠҹиғҪгҖӮ
- `ChatCoreGUI`пјҡWindows Forms зӨәдҫӢзЁӢеәҸпјҢз”ЁдәҺй…ҚзҪ®е№¶зӨәиҢғдҪҝз”ЁгҖӮ
- `ChatCoreTester`пјҡз”ЁдәҺжүӢеҠЁжөӢиҜ•зҡ„зӨәдҫӢзЁӢеәҸгҖӮ
- `ChatCoreSVG`пјҡд»Һ SVG з”ҹжҲҗеӣҫеғҸзҡ„е·Ҙе…·йЎ№зӣ®гҖӮ

## и®ҫи®ЎжЁЎејҸ
- **жңҚеҠЎ/жҸҗдҫӣеҷЁжЁЎејҸ**пјҡжҜҸдёӘиҒҠеӨ©жңҚеҠЎйғҪе®һзҺ° `IChatService`пјҢе№¶з”ұеҜ№еә”зҡ„жңҚеҠЎз®ЎзҗҶеҷЁз®ЎзҗҶгҖӮ
- **дәӢд»¶й©ұеҠЁ**пјҡ`ChatServiceBase` жҸҗдҫӣеӨҡз§ҚдәӢд»¶пјҢдёҺ `DictionaryUtils` й…ҚеҗҲдҪҝз”Ёд»Ҙз®ЎзҗҶе№¶еҸ‘еӣһи°ғгҖӮ
- **дҫқиө–жіЁе…Ҙ**пјҡ`ChatCoreInstance.Create()` жһ„е»ә `ServiceProvider`пјҢй…ҚзҪ®жүҖжңүжңҚеҠЎгҖӮ
- **еӨҡжөҒзЁӢеӨ„зҗҶ**пјҡ`ChatServiceMultiplexer` иҒҡеҗҲеӨҡдёӘжңҚеҠЎпјҢе№¶еҸҜеҠЁжҖҒејҖеҗҜжҲ–е…ій—ӯгҖӮ
- **й…ҚзҪ®ж–Үд»¶**пјҡ`ConfigBase<T>` йҖҡиҝҮ `FileSystemWatcher`зӣ‘еҗ¬ж–Үд»¶жӣҙж–°пјҢиҮӘеҠЁйҮҚж–°еҠ иҪҪгҖӮ
- **ж—Ҙеҝ—и®°еҪ•**пјҡ`Logging` зӣёе…іж–Үд»¶е®ҡд№үдәҶиҮӘе®ҡд№үж—Ҙеҝ—зә§еҲ«дёҺжү“еҚ°жҺҘеҸЈгҖӮ

## д»Јз ҒйЈҺж ј
йЎ№зӣ®йҖҡиҝҮ `.editorconfig` зәҰжқҹзј–зЁӢйЈҺж јпјҡ
- зј©иҝӣдҪҝз”Ё **еҲ¶иЎЁз¬Ұ**пјҲindent_style = tabпјүпјҢе№¶д»Ҙ tab дҪҚе®ҡеӯ—иҠӮж•°гҖӮ
- жңҖеӨ§иЎҢе®Ҫдёә **200** еӯ—з¬ҰпјҢе№¶дҪҝз”Ё LF з»Ҳз«Ҝз¬ҰгҖӮ
- еҺ»жҺүиЎҢжң«з©әж јпјҢ`using` д»Ҙ System жөӢиҝҮз»„еӯҳеҶҷгҖӮ
